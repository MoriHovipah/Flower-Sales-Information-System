# Flower-Sales-Information-System

## Overview
Flower Sales Information System is a web-based application developed to
help flower shop owners manage sales transactions, inventory, and
financial reports in a structured and efficient way.

## Problem Background
Manual sales recording often leads to data inconsistency, calculation
errors, and time-consuming report generation. This system aims to
digitalize the sales process to improve data accuracy and operational
efficiency for small flower shops.

## Features
- User authentication (login & logout)
- Flower inventory management
- Sales transaction recording
- Automatic stock updates
- Sales report generation (daily, weekly, monthly)
- Export sales reports to Excel
- Print sales receipt
